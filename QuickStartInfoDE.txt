hier wie besprochen einige Info’s zu Retroshare:

Woher bekommt man RS?
	http://retroshare.sourceforge.net/downloads.html

Welche Version soll ich nehmen?
	Unter Windows die RS Portable nehmen, da einfacher zu updaten.
	http://sourceforge.net/projects/retroshare/files/RetroShare/0.5.3c/RetroShare_0.5.3c_5190_portable.zip/download

5min Quickinstall?
	RS Portable runterladen und entpacken
	Retroshare.exe starten un profil anlegen (Nickname, fakemail, pwd, location)
	Eignen key mit vorher eingegeben Pwd unterschreiben
	Optionen --> Sicherheit --> Eigenen Key in Datei speichern
	Eigenen Key per Mail, Skype whatever an Freund senden
	Freund hinzufügen, indem man den von diesem erhaltenen Key empfängt, dann Freunde  Freund hinzufügen  Key von Freund eingeben / auswählen
	Optional: Optionen  Server  Netzwerkkonfiguration  Portfreigabe per Hand (<< Port in Router und Firewall weiterleiten)
	Fertisch, viel Spass beim sicheren kommunizieren =)
	(In der Taskleiste kann man den Messager Öffnen, dann verhält sich RS ähnlich wie Skype)
